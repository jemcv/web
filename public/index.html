Hey! I'm Jem, Welcome to my personal website. This is where I'm going to put my codes and my posts. My plan is to post my codes here and to write some things then post it here.

I use arch btw. From what I've experienced setting it up, Arch Linux  is an operating system which you need to configure it from scratch to have a working desktop. I'm currently using it for one month and decided to use a window manager. The window manager I use is dwm. "dwm is a dynamic window manager for X. It manages windows in tiled, monocle and floating layouts. All of the layouts can be applied dynamically, optimising the environment for the application in use and the task performed" copied from the suckless website. >_<

My dwm build currently has 5 patches to improve my workflow. I just started using arch and I know I still have a lot of things to explore in this cave. This makes me more excited to tinker things and explore technology especially in the FOSS world. I'm writing this in 08-08-2022 and probably rewrote this after a year. :)

The other tools I use is a menu also from the suckless.org called dmenu, basically an app launcher, I haven't patched it yet but I added pywal to it. Pywal is used for generating colors from an image/background. For the status bar, I'm using dwmblocks where you can use shell scripts to retrieve information from your os and the bar will display it.
